## ?? Security Architecture

### Authentication Flow

```
1. User Login
   ?
2. AuthController validates credentials
   ?
3. Generate JWT Token
   ?
4. Return Token to Client
   ?
5. Client stores Token (localStorage/sessionStorage)
   ?
6. Subsequent requests include Token in Header
   ?
7. Middleware validates Token
   ?
8. Authorize based on Roles/Permissions
```

